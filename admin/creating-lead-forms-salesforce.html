在Salesforce中創建潛在客戶表單

{{ page.description }}

Brightcove Campaign提供了創建銷售線索表格的功能，該功能使您可以在視頻播放期間捕獲觀眾信息。表單使您能夠包括所需的任何字段（可見或隱藏），還使您能夠利用諸如逐步分析（如果有）之類的功能。潛在客戶數據將發送到Salesforce並由其進行處理。

在Salesforce中創建表單

Salesforce中的Web-to-Lead功能可用於從網站訪問者生成潛在客戶信息。這些表格也可以顯示在Brightcove Campaign視頻播放器中。下面的步驟將創建一個簡單的表單來捕獲名字，姓氏，電子郵件地址，公司，城市和州/省。有關網絡領先功能的完整詳細信息，請參見Salesforce文檔。

  1. 登錄到您的Salesforce帳戶。
  2. 點擊設置（齒輪圖標）。
  3. 進入Web-to-Lead在快速查找框中，然後單擊網絡對領導。
  4. 點擊創建Web到引線表格。
  5. 選擇要包含在“Web到鉛"表單中的字段。使用添加和刪除箭頭在可用字段列表和選定字段列表，然後使用向上和向下箭頭更改表單上字段的順序。
  6. 進入一個返回網址 .在此示例中，“返回URL"將設置為http://www.brightcove.com。當表單用作Brightcove Campaign中的潛在客戶表單時，將忽略此值。
  7. 點擊產生。將顯示HTML表單。

複製和編輯HTML表單

要在Brightcove Campaign中使用該表單，您將需要復制HTML表單。另外，您可能需要對錶單進行一些基本的樣式設置。在下面的步驟中，我們將表單HTML複製到HTML編輯器，然後對錶單進行一些樣式更改。

默認情況下，顯示表單時，表單將在播放器上方的透明背景上顯示。使用CSS設置背景顏色。

  1. 將HTML複製到剪貼板。
  2. 點擊完成的。
  3. 將表單HTML粘貼到您選擇的HTML編輯器中。您可能需要使用編輯器的代碼格式化功能來格式化代碼，以便於閱讀。
  4. 使用您的 HTML 編輯器，添加以下 CSS 以將背景顏色更改為灰色。
    <style>
    body {
    background-color: rgba(204,204,204,0.9);
    margin: 0;
    padding: 20px;
    height: 100%;
    }
    </style>
  5. （可選）要確定 Brightcove Campaign 生成了哪些潛在客戶，可以在<form>標籤：
    <input type="hidden" name="lead_source" value="Brightcove Campaign">
  6. 將HTML複製到剪貼板。

在Salesforce中驗證表單數據

使用自定義潛在客戶表單捕獲的潛在客戶將作為潛在客戶添加到您的Salesforce帳戶中。
