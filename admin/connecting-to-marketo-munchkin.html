使用 Munchkin 連接到 Marketo

{{ page.description }}

可以將Brightcove Campaign配置為與Marketo集成。配置集成後，使用Brightcove Campaign發布的視頻的視頻參與度數據將同步到Marketo。進入Marketo後，該數據可用於細分，廣告系列邏輯，報告，潛在客戶評分和個性化通信。

注意：有關如何使用 REST API 配置 Brightcove Campaign 以與 Marketo 一起使用的信息，請參閱使用 REST API 連接到 Marketo .

Marketo集成的一些關鍵功能包括：

  • 資料格式 -視頻觀看數據以頁面瀏覽或虛擬Web訪問的形式傳遞給Marketo。網頁訪問事件包含指示觀看者參與度的查詢參數。複製到Marketo的數據如下：
    • 影片名稱
    • 視訊識別碼
    • Brightcove帳戶ID
    • 推薦頁面的頁面URL
    • Brightcove玩家編號
    • % Watched
  • 數據傳輸 -查看數據會立即發送到Marketo，但是，Marketo平台中的某些報告可能需要一些時間才能刷新
要將Brightcove Campaign與Marketo Munchkin一起使用，您必須具有啟用了Munchkin API的Marketo帳戶-您將需要Munchkin帳戶ID（請參見進入Marketo Munchkin以下）。具有Munchkin用戶角色的用戶應該能夠訪問這些值。

進入Marketo Munchkin

在配置與Brightcove Campaign的集成時，將提示您輸入Marketo munchkin。請按照以下步驟訪問此值。

  1. 登錄到您的Marketo帳戶。
  2. 點擊行政在頂部導航中。
  3. 在左側導航中，導航至行政 > 一體化 > 蒙奇金。
  4. 複製Munchkin帳戶ID並將其保存在安全的地方。配置集成時將需要它。

設置連接

請按照以下步驟在Brightcove Campaign和Marketo之間建立連接。

建立Brightcove Campaign和Marketo之間的聯繫

  1. 登入布萊特灣戰役
  2. 在導覽標題中，按一下齒輪圖示 (） 進而地圖連接 .將顯示當前連接列表（如果有）。
  3. 點擊添加新連接 .
  4. 選擇市場部從下拉列表中。
  5. 選擇高音量（蒙克金）集成類型。
  6. 輸入您的Marketo帳戶ID。這是從Marketo複製的Munchkin帳戶ID。
  7. 點擊節省 .

現在已建立與Marketo的連接並可以使用。默認情況下，連接狀態將會啟用 .這連接狀態確定是否將視頻參與度數據發送到Marketo：

  • 已啟用 -收集數據並將其發送到Marketo
  • 殘障人士 -未收集數據（禁用集成）

點擊節省保存更改。要刪除集成，請單擊刪除。
