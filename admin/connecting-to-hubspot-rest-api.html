使用 REST API 連接到 HubSpot

{{ page.description }}

可以將Brightcove Campaign配置為與HubSpot集成。配置集成後，使用Brightcove Campaign發布的視頻的視頻參與度數據將同步到HubSpot。進入HubSpot後，此數據可用於細分，廣告系列邏輯，報告，潛在客戶評分和個性化通信。

注意：有關如何使用客戶端 API 配置 Brightcove Campaign 以與 HubSpot 一起使用的信息，請參閱使用客戶端 API 連接到 HubSpot .

Brightcove Campaign-HubSpot集成的一些關鍵功能包括：

  • 數據格式 - 查看數據作為自定義交付時間線事件類型 .複製到HubSpot的數據如下：
    • 影片名稱
    • 視訊識別碼
    • Brightcove帳戶ID
    • 推薦頁面的頁面URL
    • Brightcove玩家編號
    • 觀看百分比
  • 數據傳輸 -使用自動過程將查看數據發送到HubSpot；但是，HubSpot平台中的某些報告可能需要一些時間才能刷新

設置連接

請按照以下步驟設置與HubSpot的連接。

注意：連接過程將在瀏覽器中打開彈出窗口。您應該在設置連接之前禁用任何瀏覽器彈出窗口阻止器。

建立Brightcove Campaign和HubSpot之間的連接

  1. 登入布萊特灣戰役
  2. 在導覽標題中，按一下齒輪圖示 (） 進而地圖連接 .將顯示當前連接列表（如果有）。
  3. 點擊添加新連接 .
  4. 選擇HubSpot從下拉列表中。
  5. 選擇標準（REST API）集成類型。
  6. 點擊連接到 HubSpot .
  7. 系統將提示您登錄HubSpot。輸入您的電子郵件地址和密碼 .點擊登錄 .
  8. 如果登錄成功，可能會提示您允許Brightcove 視頻連接訪問您的帳戶。點擊授予訪問權限 .

至HubSpot的連接現已建立並可以使用。默認情況下，連接狀態將會啟用 .這連接狀態確定是否將視頻參與度數據發送到Marketo：

  • 已啟用 -收集數據並將其發送到HubSpot
  • 殘障人士 -未收集數據（禁用集成）

點擊保存保存更改。要刪除集成，請單擊刪除。
