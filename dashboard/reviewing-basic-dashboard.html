查看基本儀表板

{{ page.description }}

“基本信息中心"基於觀看的平均百分比提供了效果最好的視頻的概述。顯示視頻印象，視頻開始，平均觀看百分比和平均觀看時間的圖形表示。

啟用視頻的“視頻內容類型"元數據字段時，Brightcove Campaign將顯示高級儀表板。先進的信息中心為每個視頻提供了視頻基準評分（VBS），可將您的視頻的性能與其他相同類型的類似視頻的性能進行比較。

注意：Brightcove 建議啟用視頻內容類型字段，然後使用高級儀表板。

要查看“儀表板"頁面，請單擊儀錶盤在導航標題中。

選擇時間跨度

若要指定儀表板的時間週期，您可以按一下快速日期或使用行事曆控制項。

使用快速日期

頁面頂端有一系列快速日期。可用的快速日期如下：

  • 今天
  • 昨天
  • 最近7天
  • 最近30天
  • 這個月 -迄今為止的日曆月-例如，如果今天是3月17日，則此報告將顯示3月1日-3月17日
  • 上個月 -前一個日曆月-例如，如果今天是2月14日，則此報告將顯示1月1日-1月31日。

按一下快速日期以更新儀表板。

使用行事曆控制項

您也可以按一下行事曆控制項來選取開始和結束日期。

點擊申請更新儀表板。

報告時區

使用帳戶的報告時區顯示儀表板中的數據。時區顯示在日曆控件下方。

報告時區是用來計算報告資料的日期界限。變更報告時區設定只會影響未來的資料，而且不會追溯套用變更。如果更改此設置，則在查看應用更改的日期範圍時，數據中可能會出現平坦點或峰值。對該設置的更新可能不會立即生效，並且分析數據可能會繼續引用舊設置，直到系統處理更改為止。

報告時區可以配置為帳號設定。

儀表板上使用的術語

在查看高級儀表板之前，請閱讀以下視頻術語，這將對您有所幫助。

  • 視頻開始 -視頻開始播放的次數
  • 影片曝光 -視頻加載並準備好進行互動的次數
  • 平均觀看百分比 -平均觀看時間/視頻時長
  • 平均觀看時間 -總觀看時間/開始時間

查看儀錶盤分析

頁面的正文將顯示效果最佳的視頻的分析數據。這最佳表演影片列表將顯示平均觀看率最高的前五部視頻。該列表將包含：

  • 視訊名稱
  • 簡短的介紹
  • 發佈日期
  • 平均觀看百分比

在列表中選擇一個視頻將顯示該視頻的基本分析。將鼠標懸停在圖形上將顯示確切的數據值。

點擊此視頻的更多分析將打開分析工具頁面，其中包含所選視頻的詳細分析。
