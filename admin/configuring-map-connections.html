配置Marketing Automation平台連接

{{ page.description }}

Brightcove 運動™可以配置為與流行的營銷自動化平台 (MAP) 集成，例如 HubSpot、Eloqua 和 Marketo。配置MAP集成後，使用Brightcove Campaign發布的視頻的視頻參與數據將同步到所選的營銷自動化平台。

注意：您必須是帳戶管理員才能存取管理功能表。
注意：連接過程將在瀏覽器中打開彈出窗口。您應該在設置連接之前禁用任何瀏覽器彈出窗口阻止器。

要配置與營銷自動化平台的集成，請按照下列步驟操作：

  1. 登入布萊特灣戰役
  2. 在導覽標題中，按一下齒輪圖示 (），然後映射連接。將顯示當前連接列表（如果有）。
  3. 點擊添加新連接。
  4. 選擇連接類型。請參閱適當的主題以獲取有關連接到該平台的信息。
    • 埃洛誇
    • Marketo（ REST API或者蒙奇金）
    • HubSpot（ REST API或者客戶端API）
    • 銷售隊伍
