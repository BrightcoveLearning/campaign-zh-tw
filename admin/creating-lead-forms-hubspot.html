在HubSpot中創建銷售線索表格

{{ page.description }}

Brightcove Campaign提供了創建銷售線索表格的功能，該功能使您可以在視頻播放期間捕獲觀眾信息。表單使您能夠包括所需的任何字段（可見或隱藏），還使您能夠利用諸如逐步分析（如果有）之類的功能。潛在客戶數據將發送到HubSpot並由HubSpot處理。

在HubSpot中創建表單

HubSpot提供了創建可用於捕獲查看器信息的表單的功能。在Brightcove Campaign中創建銷售線索表單時，將使用生成的HTML表單。然後可以將捕獲的數據用於自定義將來的通信。下面的步驟將創建一個簡單的表單來捕獲名字，姓氏和電子郵件地址。有關HubSpot表單的完整詳細信息，請參見HubSpot文檔。

  1. 登錄到您的HubSpot帳戶。
  2. 點擊營銷 > 形式 .如果您使用的是“經典"菜單，請單擊聯絡人 > 形式。
  3. 點擊建立表格。默認情況下，該表單將包含一個電子郵件場和提交按鈕。
  4. 確保正規表格類型被選擇。點擊下一個。
  5. 確保空白模板被選中。點擊開始。
  6. 單擊標題中的默認名稱，然後為表單命名。按進入保存更改。
  7. 頁面的左側包含可以添加到表單的字段列表。拖放名和姓您想要在頁面右側表單上的字段。
  8. 要設置必填字段，將鼠標懸停在該字段上，然後點擊更多>必填。必填字段在字段標籤旁邊顯示*。
  9. 點擊選項頁面頂部的鏈接。
  10. 為了訪客提交此表格後應該怎麼辦， 點擊顯示感謝信息。保留默認消息，因為它將永遠不會顯示。
  11. 點擊樣式和預覽頁面頂部的鏈接。默認情況下，桌面預覽將出現。
  12. 點擊發布在頁面頂部以發布對錶單所做的更改。

複製和編輯表單嵌入代碼

要在Brightcove Campaign中使用表格，您將需要復制和編輯表格嵌入代碼。另外，您可能需要對錶單進行一些基本的樣式設置。在下面的步驟中，我們將表單嵌入代碼複製到HTML編輯器，然後對錶單進行一些樣式更改。

  1. 點擊分享在頁面的右上方。
  2. 點擊嵌入代碼然後複製將嵌入代碼複製到剪貼板。
  3. 將嵌入的代碼粘貼到您選擇的HTML編輯器中。
  4. 嵌入代碼需要進行一些小的更改，因此Brightcove Campaign知道在提交表單時將其關閉。找到具有 formId:在行末添加逗號。
  5. 將此行添加到 formId:線。 
    onFormSubmit: bcLeadForm.submit
  6. 默認情況下，當窗體顯示在Brightcove Player內部時，窗體將顯示在播放器上方的透明背景上。將以下 CSS 添加到嵌入代碼以將背景顏色更改為灰色。
    <style>
    body {
    background-color: rgba(204,204,204,0.9);
    margin: 0;
    padding: 20px;
    height: 100%;
    }
    </style>

在HubSpot中驗證表單數據

使用自定義潛在客戶表單捕獲的潛在客戶將作為聯繫人添加到您的HubSpot帳戶中。查看潛在客戶表單捕獲的數據的一種簡單方法是在HubSpot中創建智能列表。

  1. 登錄到您的HubSpot帳戶。
  2. 點擊營銷>表格。
  3. 當您將鼠標懸停在表格上時，動作菜單將會出現。點擊動作>查看提交。表單提交的列表將出現。
  4. 當您將鼠標懸停在一行上時，單擊查看提交查看表單數據。
