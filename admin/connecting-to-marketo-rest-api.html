使用 REST API 連接到 Marketo

{{ page.description }}

可以將Brightcove Campaign配置為與Marketo集成。配置集成後，使用Brightcove Campaign發布的視頻的視頻參與度數據將同步到Marketo。進入Marketo後，該數據可用於細分，廣告系列邏輯，報告，潛在客戶評分和個性化通信。

注意：有關如何使用 Munchkin ID 配置 Brightcove Campaign 以與 Marketo 一起使用的信息，請參閱使用 Munchkin 連接到 Marketo .

要求

要將Brightcove Campaign與Marketo REST集成一起使用，您必須：

  • 在Marketo中創建一個有權訪問所有API權限的角色
  • 創建啟動點服務

創建僅API的用戶和LaunchPoint服務

在調用Marketo REST API之前，您需要創建一個具有對所有API的訪問權限的新API角色，創建一個僅API的用戶，然後創建一個唯一標識您的客戶端應用程序的服務。這些任務概述了完成這些任務的步驟。 Marketo文檔。

設置連接

完成Marketo端的配置步驟後，請按照以下步驟在Brightcove Campaign和Marketo之間建立連接。

建立Brightcove Campaign和Marketo之間的聯繫

  1. 登入布萊特灣戰役
  2. 在導覽標題中，按一下齒輪圖示 (） 進而地圖連接 .將顯示當前連接列表（如果有）。
  3. 點擊添加新連接 .
  4. 選擇市場部從下拉列表中。
  5. 選擇標準（REST API）集成類型。您將需要從您的Marketo帳戶中粘貼幾個值。請注意，您必須以Marketo管理員身份登錄才能完成以下步驟。
  6. 輸入您的Marketo 帳戶 ID .要獲得此值，請在Marketo中單擊行政 > 蒙奇金。
  7. 輸入您的Marketo REST API端點URL。要獲得此值，請在Marketo中單擊行政 > 網頁服務。複製終點在裡面REST API部分。
  8. 輸入您的Marketo REST API身份URL。要獲得此值，請在Marketo中單擊行政 > 網頁服務。
  9. 進入Marketo LaunchPoint客戶端ID。要獲得此值，請在Marketo中單擊行政 > LaunchPoint然後點擊查看詳情您之前在此設置中創建的LaunchPoint的鏈接。
  10. 進入Marketo LaunchPoint客戶機密。要獲得此值，請在Marketo中單擊行政 > LaunchPoint然後點擊查看詳情您之前在此設置中創建的LaunchPoint的鏈接。
  11. 點擊節省 .

現在已建立與Marketo的連接並可以使用。默認情況下，連接狀態將會啟用 .這連接狀態確定是否將視頻參與度數據發送到Marketo：

  • 已啟用 -收集數據並將其發送到Marketo
  • 殘障人士 -未收集數據（禁用集成）

點擊節省保存更改。要刪除集成，請單擊刪除。
